How Salesforce will call the service
Apex Callout Example
Salesforce → Adserve API


Http http = new Http();
HttpRequest req = new HttpRequest();
req.setEndpoint('https://your-domain.com/recommend');
req.setMethod('POST');
req.setBody(JSON.serialize(contextMap));
req.setHeader('Content-Type','application/json');

HttpResponse res = http.send(req);
RecommendationResponse r = (RecommendationResponse) JSON.deserialize(res.getBody(), RecommendationResponse.class);



Salesforce receives:
{
  "served_policy": "P1",
  "served_item": "Milk 1L",
  "probability": 0.24,
  "request_id": "p1-97d2a5e3"
}


Then when a purchase happens, Salesforce sends:
{
  "request_id": "p1-97d2a5e3",
  "purchased_item": "Milk 1L",
  "revenue": 2.99
}
